Welcome to GeoJS

GeoJS is a javascript library for visualizing geospatial data in a browser. Its flexible API provides users with the ability to combine multiple visualizations drawn in WebGL, canvas, and SVG into a single dynamic map.

What can GeoJS do for you? See the examples below.

GeoJSON file reader

Shows how to use the included geoJSON reader, and how to use the properties to adjust the style of the features. In this example, you can edit the json file while the features on the map update in response. You can also drop a new geoJSON file onto the text editor to load it.

Raw layer interface

This example demonstrates drawing directly into GeoJS layers without using the feature API. Layers come in two flavors, fixed and moving. Fixed layers remain in a fixed screen position, while moving layers remain fixed relative to the map. In addition, you may choose to rescale one or more properties in moving layers.

Feature legends

This example shows how to create a legend to describe feature styles on a map.

Open street map layer

Basic map rendering tiles from an OpenStreetMap tile server. The base url for the tile server is given at initialization to provide a custom appearance.

Mouse picking for points and lines

This example demonstrates how to attach handlers to various mouse interactions that allow the users to select features. Try moving the mouse over the features and clicking. Also try shift-click and draging to select a region

Point features

This example shows how to add styled point features to a map. It also demonstrates attaching event handlers to add interactivity and combine events from multiple layers. Notice how GeoJS events propagate through features and layers.

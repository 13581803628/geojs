Geojs feature demo

This is an iteractive demo that shows off the feature and style API of GeoJS. You can draw one the map by selecting a feature type from the menu. A dialog will pop up allowing you to customize the style of the new features. When you are ready to start drawing click the "Create" button and follow the directions in the lower right panel. When you are finished, click the button in the menu, and the features will be added to the map using the GeoJS feature API.

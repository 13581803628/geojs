Class: map

geo.map

new map(node, gcsopt, maxZoomopt, ingcsopt, unitsPerPixelopt, maxBoundsnullable, zoomopt, centernullable, widthnullable, heightnullable, minopt, maxopt, discreteZoomopt, cameranullable, interactornullable, clocknullable, autoResizeopt, clampBoundsXopt, clampBoundsYopt, clampZoomopt) → {geo.map}

Creates a new map object Map coordinates for default world map, where c = half circumference at equator in meters, o = origin: (-c, c) + o (c, c) + o (center.x, center.y) + o <-- center of viewport (-c, -c) + o (c, -c) + o

bounds(bdsopt, gcsopt) → {geo.geoBounds}

Get/set the locations of the current map corners as latitudes/longitudes. When provided the argument should be an object containing the keys lowerLeft and upperRight declaring the desired new map bounds. The new bounds will contain at least the min/max lat/lngs provided. In any case, the actual new bounds will be returned by this function.
Parameters:
Name Type Attributes Description
bds geo.geoBounds <optional>
 The requested map bounds
gcs string | geo.transform <optional>
 undefined to use the interface gcs, null to use the map gcs, or any other transform. If setting the bounds, they are converted from this gcs to the map projection. The returned bounds are converted from the map projection to this gcs.

updateAttribution() → {this}

Update the attribution notice displayed on the bottom right corner of the map. The content of this notice is managed by individual layers. This method queries all of the visible layers and joins the individual attribution notices into a single element. By default, this method is called on each of the following events: * geo.event.layerAdd * geo.event.layerRemove In addition, layers should call this method when their own attribution notices has changed. Users, in general, should not need to call this.

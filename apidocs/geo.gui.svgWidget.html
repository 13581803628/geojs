Class: svgWidget

geo.gui.svgWidget

new svgWidget() → {geo.gui.svgWidget}

Create a new instance of class geo.gui.svgWidget Due to the nature of d3 creating DOM elements as it inserts them, calls to appendChild don't appear in this widget. The canvas of an svgWidget always refers to the actual svg element. The parentCanvas can refer to another widgets svg element, dom element, or the UI layers dom element. See geo.gui.widget#parentCanvas.
